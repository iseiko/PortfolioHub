# PortfolioHUB

## Sobre
Repositório destinado aos meus projetos acadêmicos e pessoais. Contém o projeto do site-pessoal usado como entrega da disciplina (PortfolioHUB).


## Projetos principais
- projetos-academicos/site-pessoal — Site pessoal simples (HTML, CSS, JS)
- Big O analysis — análise de complexidade


## Ver o site localmente
1. Clone o repositório:

   git clone https://github.com/iseiko/PortfolioHub.git

2. Acesse a pasta do site:

   cd PortfolioHub/projetos-academicos/site-pessoal

3. Abra o arquivo `index.html` no navegador ou use uma extensão como Live Server no VSCode.


## Ver o site online (GitHub Pages)
Após habilitar o GitHub Pages (Settings → Pages → Branch: `main`, Folder: `/ (root)`), o site ficará disponível em uma URL do tipo:

https://iseiko.github.io/PortfolioHub/

Cole a URL aqui quando o deploy terminar.
