# Site Pessoal - PortfolioHUB

## Sobre o Projeto

Este projeto foi desenvolvido como parte da atividade **PortfolioHUB**, com o objetivo de criar um portfólio profissional utilizando Git e GitHub.

O site apresenta informações pessoais, projetos desenvolvidos e formas de contato, servindo como uma página inicial para o portfólio acadêmico e profissional.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

---

## Estrutura do Projeto

```text
site-pessoal/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

---

## Funcionalidades mínimas

- Links sociais para GitHub e LinkedIn.
- E-mail com link `mailto`.
- Estilos básicos para a seção de links sociais.
- Responsividade inicial para telas menores via media query.

---

## Como executar

1. Entre na pasta `projetos-academicos/site-pessoal`.
2. Abra o arquivo `index.html` no navegador.

---

## Publicação (GitHub Pages)

Após o merge na `main`, ativar em:

- **Settings → Pages → Branch: main**
- **Folder: / (root)**

URL esperada após deploy:

- `https://iseiko.github.io/PortfolioHub/`
