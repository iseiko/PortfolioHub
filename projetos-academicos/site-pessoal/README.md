# Site Pessoal - PortfolioHUB

## Sobre o Projeto

Este projeto foi desenvolvido como parte da atividade **PortfolioHUB**. É um site pessoal simples (HTML/CSS/JS) criado para apresentar informações básicas, projetos e contatos.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## Estrutura do Projeto

```
site-pessoal/
│
├── index.html
├── style.css
├── script.js
└── (opcional) screenshots/
```

## Funcionalidades

- Seção "Sobre mim"
- Lista de projetos
- Contato com link mailto
- Links para GitHub e LinkedIn
- Layout responsivo básico

## Como executar localmente

1. Clone o repositório:

   git clone https://github.com/iseiko/PortfolioHub.git

2. Acesse a pasta do site:

   cd PortfolioHub/projetos-academicos/site-pessoal

3. Abra `index.html` no navegador ou use Live Server no VSCode.

## Ver o site online (GitHub Pages)

Após o deploy do GitHub Pages (Settings → Pages → Branch: `main`, Folder: `/ (root)`), cole a URL aqui:

https://iseiko.github.io/PortfolioHub/
